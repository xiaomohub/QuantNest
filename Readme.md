
# 🚀 QuantNest

一个基于 Java + Python 的量化投资与技术平台，致力于构建**稳定复利的决策系统**，并通过 AI Coding 提升开发效率。

---

## 🧠 项目简介

QuantNest（量巢）由一名运维开发工程师发起，结合：

- 📊 量化投资
- ⚙️ 分布式系统
- 🤖 AI Coding

目标是打造一个：

> **稳定、可扩展、可验证的量化策略与执行平台**

---

## 🏗 技术栈

### 后端
- Java 17
- Maven
- Spring Boot（规划中）

### 数据与分析
- Python（策略研究 / 数据分析）

### 运维与部署
- Docker
- Docker Compose
- Linux

### 其他
- Redis（缓存）
- MQ（异步处理）
- REST API

## ⚙️ 快速开始

### 1️⃣ 克隆项目

```bash
git clone https://github.com/你的用户名/quant-nest.git
cd quant-nest
```

---

### 2️⃣ 构建项目

```bash
mvn clean install
```

---

### 3️⃣ 运行项目

```bash
java -jar target/*.jar
```

---

## 🐳 Docker 运行（规划中）

```bash
docker build -t quant-nest .
docker-compose up -d
```
